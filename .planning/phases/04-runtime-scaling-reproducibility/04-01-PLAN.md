<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| upstream HTTP -> downloader cache | Untrusted third-party blocklist bytes enter local cache storage. |
| cache -> raw output | Cached bytes are promoted into generated raw source files consumed by the pipeline. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-01-01 | Tampering | `fetch_url()` streamed 2xx path | mitigate | Stream to cache-side temp, replace final files only after complete writes, and update state after promotion per D-01/D-02. |
| T-04-01-02 | Denial of Service | fallback cache copies | mitigate | Reuse bounded copy helpers for 304/error/timeout/exception fallback paths so cache promotion memory scales with chunk size per D-03. |
| T-04-01-03 | Denial of Service | source-size/content-type policy | accept | Phase 04 records source metrics only and deliberately avoids new hard source-size or content-type gates per D-04/D-05. |
</threat_model>
